# Images des Logos

Placez vos images de logos dans ce dossier :

- `google_logo.png` - Logo Google (recommandé: 24x24 ou 32x32 pixels)
- `x_logo.png` ou `twitter_logo.png` - Logo X/Twitter (recommandé: 24x24 ou 32x32 pixels)

## Format recommandé
- Format: PNG avec transparence
- Taille: 24x24 à 48x48 pixels
- Fond: Transparent de préférence

## Où trouver les logos officiels
- Google: https://developers.google.com/identity/branding-guidelines
- X/Twitter: https://about.twitter.com/en/who-we-are/brand-toolkit
